Overview

Download latest version at SourceForge here. You will find SourceForge project page here.

The JUnit 4 Extensions (www.junitext.org) is an OpenSource project started during writing an article about new features of JUnit 4 (See: www.ix.de, Issue 06/2006). These extensions should proove extensibility of JUnit 4 for typical use cases.

These extensions will support following features (see also ChangeLog):

  • Additional annotation for Prerequisites (@Prerequisite): Ignore tests based on conditions. Will use the org.junit.runner.manipulation.Filter
  • Additional annotation for Categories (@Category): Filter and Sort tests based on a Category. Will both use the org.junit.runner.manipulation.Filter and org.junit.runner.manipulation.Sorter
  • Annotation based runner, to be used for @RunWith(AnnotationRunner.class)
  • As requested by some JUnit users: Provide the old UI based runners from JUnit 3.8.2 as prebuilt jar file

Following features are planned in near time (see also TODO):

  • Additional runner for multithreaded tests: @RunWith(MTRunner.class)
  • Additional annotation for @Platform: Run test only based on given platform spec (OS version, JDK version, Eclipse version, ...)
  • Procide declarative test configurations (as provided by TestNG)
  • Test whether JUnit 4 extensions could be plugged into Eclipse JUnit plugins
